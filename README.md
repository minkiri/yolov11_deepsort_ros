**yolov11_deepsort_ros**

1. Git clone.
    - `git clone https://github.com/minkiri/yolov11_deepsort_ros.git`
    - `catkin_make`
    - `source devel/setup.bash`
<br>

2. Environment install
    - `conda env create -f yolov11d.yaml`
